# 🧠 Lunary OS — Инструкция запуска

## Быстрый старт (локально)

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Запусти систему
```bash
python run.py
```

Это запустит:
- ✅ API на http://localhost:8000
- ✅ Дашборд на http://localhost:8000
- ✅ Сканер на http://localhost:8000/scanner
- ✅ Telegram бот

### 3. Открой бота в Telegram
Найди своего бота и напиши `/start`

---

## Команды бота

| Команда | Действие |
|---|---|
| `остаток герметик белый` | Узнать остаток |
| `минус 3 герметик серый` | Списать продажу |
| `плюс 20 герметик черный` | Приход товара |
| `списать 2 герметик` | Ручное списание |
| `/low` | Что заканчивается |
| `/list` | Все товары |
| `/scan` | Открыть сканер штрихкода |

---

## Деплой на Railway

1. Создай аккаунт на railway.app
2. Подключи GitHub репозиторий
3. Railway автоматически запустит через Procfile

## Структура проекта

```
lunary/

├── crud.py         # Операции с БД
├── seed.py         # Начальные данные
├── requirements.txt
├── Procfile        # Для Railway
└── static/

    └── scanner.html # Сканер штрихкода
```
